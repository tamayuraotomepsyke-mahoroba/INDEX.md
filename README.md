・Human–AI 共生知性

・連続状態モデル（0123）
・責任履歴モデル（0123＋−1）

**Sustainable Intelligence Environment**

for long-term Human–AI interaction.

本研究は、長期対話における

**持続可能な知性環境
（Sustainable Intelligence Environment）**

の設計を目的とする。
